# 🥩 Steak Timer

A small pixel-style steak cooking timer built with **Streamlit**.

This web app helps you cook steak to the desired doneness using a simple animated timer and pixel-style UI.

🔗 **Live App**  
https://feyzis-steak-timer.streamlit.app

---

## ✨ Features

- Choose steak thickness
- Choose steak doneness level
- Automatic cooking time calculation
- Pixel-style steak cooking animation
- Countdown timer
- Cooking progress bar
- Alarm sound when the steak is ready
- Simple retro pixel UI

---

## 🥩 Doneness Levels

The timer supports common steak doneness levels:

- Rare
- Medium Rare
- Medium
- Medium Well
- Well Done

Cooking time is automatically adjusted based on steak thickness.

---

## 🛠 Tech Stack

- **Python**
- **Streamlit**
- **Pillow**

---

## 🚀 Run Locally

Clone the repository:

git clone https://github.com/msubasii/steak-timer.git
steak-timer

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run main.py

The app will open in your browser.
